# Smith Slay and Sell

A co-op multiplayer game in active development by the De Anza Game Development Club. 

We're excited to share that this project has been selected to be showcased at the [2026 De Anza Expo](https://deanzaexpo.org/), an event where industry leaders check out student-created projects.

## About the Game
It's a chaotic co-op multiplayer game where you have to work together to frantically forge weapons under pressure. The long-term plan is to take the weapons you craft and use them to fight off enemies, but our immediate focus is locking in the core smithing gameplay.

## Current Status
We have established a solid technical foundation. The heavy lifting on the scripting side is largely complete, and we are rapidly transitioning into building out the scenes and visual elements. 

The core systems already implemented include:
* Character controllers
* Co-op input and dynamic player spawning
* Item interaction (picking up, dropping, holding)
* Station mechanics

## Built With
* Unity
